# 🚀 MiroFish Electron - Быстрый старт

## ТРИ ШАГА ДО ГОТОВОГО ПРИЛОЖЕНИЯ

### Шаг 1️⃣: Установи зависимости (5 минут)

```powershell
cd D:\FACTORY\MiroFish

# Запусти скрипт сборки
.\build-app.ps1
```

Это автоматически:
- ✅ Проверит Node.js, Python, Ollama
- ✅ Установит все зависимости
- ✅ Соберёт Frontend и Backend
- ✅ Создаст .exe файл

### Шаг 2️⃣: Убедись что Ollama запущена (1 минута)

```powershell
# В отдельном терминале:
ollama serve

# Должно вывести:
# >>> Listening on http://127.0.0.1:11434
```

### Шаг 3️⃣: Запусти установщик

```powershell
# В папке MiroFish/build/ найди:
# MiroFish-0.1.0.exe

# Запусти его и следуй инструкциям
# Приложение установится на D:\ диск
```

---

## 🎯 ГОТОВО! 

После установки на рабочем столе появится ярлык **MiroFish**.

Кликни его и приложение запустится:
1. ✅ Backend запустится автоматически
2. ✅ Загрузится список моделей Ollama  
3. ✅ Выбери модель (например, `qwen3.5:35b`)
4. ✅ Создавай симуляции как обычно

---

## 🔄 ОБНОВЛЕНИЯ

Приложение автоматически проверяет обновления при запуске.

Если есть новая версия:
1. Уведомление появится в приложении
2. Обновление скачается в фоне
3. При следующем запуске установится

Чтобы проверить вручную: **Menu → Help → Check for Updates**

## ❌ ЧАСТЫЕ ОШИБКИ

| Ошибка | Решение |
|--------|---------|
| "Ollama не найдена" | Запусти `ollama serve` в отдельном терминале |
| "Node.js не найден" | Скачай https://nodejs.org/ (≥18.0.0) |
| "Python не найден" | Скачай https://python.org/ (3.11-3.12) |
| "Модели не загружаются" | Скачай модель: `ollama pull qwen3.5:35b` |
| "Backend не стартует" | Проверь логи в MiroFish/logs/ |
