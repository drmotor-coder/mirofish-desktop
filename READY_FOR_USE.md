## 🎮 ИСПОЛЬЗОВАНИЕ

### Первый запуск

1. **Кликни ярлык на рабочем столе**
2. **Приложение запустится** (backend запустится автоматически)
3. **Кликни кнопку ⚙️ внизу справа** (Settings)
4. **Выбери модель LLM:**
   - `qwen3.5:35b` ⭐ рекомендуется
   - `qwen3.6:35b` новая версия
   - `gemma-4-e4b` быстрая
   - Или любая другая из Ollama

5. **Начни использовать приложение как раньше!**

### Конфигурация (Settings ⚙️)

**Пользователь может менять БЕЗ КОДА:**
- Выбор LLM модели
- Backend URL
- Ollama URL
- GPU device (CUDA)
- Параметры симуляции
- LLM параметры (temperature, контекст)
- Автообновления

## 🐛 TROUBLESHOOTING

### Приложение не запускается

```powershell
# 1. Убедись что Ollama работает
ollama serve

# 2. Проверь портаhp
curl http://localhost:11434/api/tags

# 3. Посмотри логи
# %APPDATA%\MiroFish\logs\

# 4. Перезапусти приложение
```

### Модели не загружаются

```powershell
# Скачай модель вручную
ollama pull qwen3.5:35b

# Потом перезапусти MiroFish
```

### Backend не стартует

```powershell
# Проверь Python версию
python --version  # Должна быть 3.11 или 3.12

# Посмотри логи
# %APPDATA%\MiroFish\logs\
```
